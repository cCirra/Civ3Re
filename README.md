# Civ3Re

This repository contains a small fanmade remake of Civilization III built in Python. The goal is to gradually implement classic turn-based gameplay mechanics. The project is purely educational and contains no original game assets.

## Structure

- `src/` - Python source files for game logic
- `tests/` - Unit tests
- `main.py` - Example script running a tiny simulation

## Running

Use Python 3.12 or newer:

```bash
python3 main.py
```

Run tests with:

```bash
python3 -m unittest discover tests
```
